## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on Zomato restaurant data to understand restaurant trends, customer preferences, ratings, cost distribution, and ordering behavior.

The main goal is to transform raw restaurant data into meaningful business insights using data cleaning, statistical analysis, and visualizations.

---

## 🎯 Objective
The objective of this project is to analyze restaurant data and answer business questions such as:

- Which restaurants receive the highest votes?
- Do restaurants offering online ordering get better ratings?
- Does higher cost lead to better ratings?
- Which restaurant type is most common?
- How are ratings distributed?

## 🧹 Data Preprocessing
Before analysis, the dataset was cleaned using the following steps:

- Removed duplicate records
- Checked missing/null values
- Converted rating column into numeric format
- Cleaned inconsistent data entries
- Prepared dataset for visualization

## 🔍 Key Insights

### 1. Online Ordering
Most restaurants provide online ordering services.

### 2. Table Booking
Only a small percentage of restaurants offer table booking.

### 3. Ratings
Most restaurant ratings lie between **3.5 and 4.5**.

### 4. Cost vs Rating
Higher cost does not always mean better ratings.

### 5. Popular Restaurants
Restaurants with more votes are generally more popular and trusted.

### 6. Restaurant Type
Dining is the most common restaurant category.
